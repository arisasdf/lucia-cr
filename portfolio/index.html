Symmetry Sketch

MSc Software Engineering, Cardiff Uni, May 2019

Experimental Android app. Built for the User Experience module of my MSc at Cardiff University, using with MIT App Inventor 2. It was created as a beta version in order to obtain and properly process user feedback.

The app allows the user to sketch and will simultaneously rotate or mirror the user's the strokes in order to create, two, four or eight lines, with the option to pick a color for the line. Some of the most popularly requested features were saving the drawings, more colors, a paint bucket and erasing.

The app is currently not available in the Play store, but you can download the exported AI2 project to run directly from the AI2 website.
